Задание состоит из нескольких шагов:

1. Установите EditorConfig для вашего любимого редактора.

2. Посмотрите видеокурс по работе с Git и GitHub (если до этого ещё не посмотрели).

3. Сгенерируйте на рабочем компьютере SSH-ключ и добавьте его в свой аккаунт на GitHub. Информацию о том, как это сделать, читайте в нашей статье.

4. Сделайте форк личного проекта, ссылка на который есть в правой колонке интерфейса ученика. Если ссылки нет, то надо немного подождать, пока наш робот создаст репозиторий вашего проекта.

5. Затем клонируйте свой репозиторий (форк) личного проекта к себе на компьютер по SSH.

6. Создайте ветку module1-task1 в склонированном проекте.

7. В этой ветке внесите изменения в файл Readme.md (с помощью вашего любимого редактора): укажите своего наставника.

8. Создайте пулреквест из ветки module1-task1 вашего форка в мастер-репозиторий.

9. Нажмите кнопку «Найти пулреквест», дождитесь пока наш робот найдёт его и привяжет к заданию.

10. Дополнительно вы можете собрать ваш проект и разместить его в интернете. Для этого нажмите кнопку «Опубликовать».

11. Нажмите зелёную кнопку «Задание готово», чтобы отправить задание на проверку.

Если наставник попросит внести дополнительные изменения, то их нужно вносить в форке, коммитить новые изменения и синхронизовать ваш форк. Новый пулреквест создавать не нужно. Все изменения автоматически появятся в уже созданном пулреквесте.
